# pico-vision-examples

Just some fun sample games and examples I made for the Pico Vision.

## Projects

- `tetris.py` – A classic falling-block Tetris game.
- `invaders.py` – A retro-style Space Invaders shooter.
- `pong.py` – A basic head-to-head Pong match.
- `matrix.py` – A Matrix-inspired cascading text effect.
